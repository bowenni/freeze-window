Insert one cron expression (http://en.wikipedia.org/wiki/Cron) per line to add freeze windows in UTC time 
  Cron expression supported is as follow:
  
  # * * * * * *
  # │ │ │ │ │ |
  # │ │ │ │ │ └── (Optional)  year (1970-2099)
  # │ │ │ │ └───── day of week (0 - 6) (0 to 6 are Sunday to Saturday)
  # │ │ │ └────────── month (1 - 12)
  # │ │ └─────────────── day of month (1 - 31)
  # │ └──────────────────── hour (0 - 23)
  # └───────────────────────── min (0 - 59)
  
  No support for seconds.
  
  No support for specifying both a day-of-week and a day-of-month value (you'll need to use the '?' character in one of these fields)
  
  Examples:
    * 9-17 ? * 1-5  (Build does not run between 9am to 5pm from Monday to Friday UTC time)
    * 8-12 ? * 6-7  (Build does not run between 8am to 12pm on Saturday and Sunday UTC time)
    * 14-20 1 * ?   (Build does not run between 2pm to 8pm on 1st of every month UTC time)
